## 🚀 كيفية الاستخدام:

### في HTML:
```html
<!-- إضافة ملفات CSS -->
<link rel="stylesheet" href="assets/css/modern-design.css">

<!-- إضافة ملفات JS -->
<script src="assets/js/modern-animations.js"></script>
```

### استخدام الفئات:
```html
<!-- بطاقة حديثة -->
<div class="modern-card animate-on-scroll">
    <!-- المحتوى -->
</div>

<!-- زر حديث -->
<button class="modern-btn modern-btn-primary">
    <i class="fas fa-shopping-cart"></i>
    إضافة للسلة
</button>

<!-- أيقونة حديثة -->
<div class="modern-icon">
    <i class="fas fa-heart"></i>
</div>
```

### استخدام JavaScript:
```javascript
// إظهار إشعار
showModernNotification('تم الحفظ بنجاح!', 'success');

// عداد متحرك
animateCounter(element, 1000, 2000);

// تأثير الكتابة
typeWriter(element, 'نص للكتابة', 50);
```
