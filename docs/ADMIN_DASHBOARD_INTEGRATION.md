# ربط لوحة الإدارة الشاملة بالـ Backend

## ✅ ما تم إنجازه:

### 1. إنشاء API Client موحد (`js/api.js`)
- عميل API موحد للاتصال بالـ Backend
- إدارة Token تلقائياً
- دعم جميع الـ APIs: Products, Orders, Contests, Did You Know, Stats, Notifications

### 2. ربط لوحة الإدارة (`js/admin-dashboard-integration.js`)
- تحميل الإحصائيات من الـ Backend
- تحميل الطلبات وتحديثها تلقائياً
- تحميل المنتجات وعرضها
- تحميل المسابقات النشطة
- تحميل بيانات العملاء (من الطلبات)
- تحديث تلقائي كل 30 ثانية

### 3. ربط الصفحة الرئيسية (`index.html`)
- تحميل المنتجات من الـ API بدلاً من البيانات الثابتة
- عرض المنتجات المميزة (featured) تلقائياً
- تحديث تلقائي عند إضافة منتجات جديدة من لوحة الإدارة

### 4. تحديث لوحة الإدارة (`admin-dashboard-tabs.html`)
- إضافة IDs للعناصر التي تحتاج تحديث
- ربط scripts للـ API Integration
- دعم التحديث التلقائي للبيانات

## 📋 الميزات:

### ✅ لوحة الإدارة الشاملة:
1. **النظرة العامة (Overview)**
   - إحصائيات المبيعات الشهرية (من الـ API)
   - عدد الطلبات قيد الشحن (من الـ API)
   - إجمالي العملاء (من الـ API)
   - معدل النمو

2. **إدارة المنتجات**
   - عرض جميع المنتجات من قاعدة البيانات
   - إمكانية التعديل والحذف
   - تحديث تلقائي عند إضافة منتجات جديدة

3. **إدارة الطلبات**
   - عرض جميع الطلبات من قاعدة البيانات
   - عرض حالة كل طلب
   - إمكانية فتح تفاصيل الطلب

4. **إدارة العملاء**
   - عرض كبار العملاء (حساب تلقائي من الطلبات)
   - عدد الطلبات لكل عميل
   - إجمالي المشتريات

5. **إدارة المسابقات**
   - عرض المسابقات النشطة من قاعدة البيانات
   - عدد المشاركين
   - حالة المسابقة

### ✅ الصفحة الرئيسية:
- تحميل المنتجات المميزة من الـ API
- عرض تلقائي عند إضافة منتجات جديدة
- تحديث تلقائي كل 10 ثواني

## 🔄 كيف يعمل:

### 1. عند فتح لوحة الإدارة:
```javascript
// يتم تحميل البيانات تلقائياً
loadDashboardData() {
    - loadStats()        // الإحصائيات
    - loadOrders()       // الطلبات
    - loadProducts()     // المنتجات
    - loadContests()     // المسابقات
    - loadCustomers()    // العملاء
}
```

### 2. التحديث التلقائي:
- كل 30 ثانية: تحديث جميع البيانات في لوحة الإدارة
- كل 10 ثواني: تحديث المنتجات في الصفحة الرئيسية

### 3. عند إضافة منتج جديد:
1. تذهب إلى لوحة الإدارة → إدارة المنتجات → إضافة منتج جديد
2. يتم حفظ المنتج في قاعدة البيانات
3. تلقائياً يظهر في:
   - لوحة الإدارة (في جدول المنتجات)
   - الصفحة الرئيسية (إذا كان featured)

## 📝 ملاحظات مهمة:

### ⚠️ Authentication:
- يجب تسجيل الدخول أولاً للحصول على Token
- Token يتم حفظه في `localStorage`
- جميع طلبات الـ API تتضمن Token تلقائياً

### ⚠️ API Endpoints المطلوبة:
- `/api/products` - جلب المنتجات
- `/api/orders` - جلب الطلبات
- `/api/contests` - جلب المسابقات
- `/api/stats` - جلب الإحصائيات (يتطلب authentication)

### ⚠️ CORS:
- تأكد من تفعيل CORS في الـ Backend
- يجب أن يسمح الـ Backend بطلبات من نفس الـ Origin
